The main file is the icp.py which can be run simply in python.
There are several arguments that can be tweaked like changing the sampling method, number of samples.

The current file produces some tables found in the second section. Other results can be found by using other sampling methods.


Results of Assigment 3.1 and 3.2 can be found using human_reconstruction and human_rec_merge respectively.

Some results use parts of the dataset and different step sizes which can be changed with variables N and S in these functions.
